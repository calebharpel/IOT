Exercise 1-Install Nagios

ref: Linux and Ubuntu Article

  1. These steps are performed on the Raspberry Pi.
  2. Open a Terminal window and enter the following commands to ensure your Raspberry Pi is up to date
    sudo apt-get update
    sudo apt-get full-upgrade
  3. In the Terminal window enter the following command to install the Apache2 webserver:
    sudo apt install nagios3
  4. When prompted set the nagiosadmin password to Nagio$

    Nagios Admin

  5. After the Nagios3 install is complete, point your browser to the url: 127.0.0.1/nagios3
    Enter username nagiosadmin and password Nagio$

    Nagios Admin

  6. If the webpage does not appear, troubleshoot and resolve any issues.
Exercise 2-Examine Nagios3
  1. These steps are performed on your Raspberry Pi.
  2. Open the browser and go to: 127.0.0.1/nagios3 (if it is not already running)
  3. The home page contains links to the currrent status of your Raspberry Pi, and getting started.
  4. In the left column, under the Current Status section, click the 'Services' link and examine the details.

    Nagios Services

  5. To find out more about Nagios visit this web page:
    https://raspberrytips.com/nagios-raspberry-pi/
    Note that installing Nagios 4+ is more difficult, but doable.
    It is good to install the snmpd on each network host so that you can monitor it with Nagios.
    Then learn to add the new host to the Nagios configuration file.
