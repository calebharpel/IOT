Setting Up a micro:bit




60 Points
Learning Objectives

These instructions will show you how to download and create the operating system for a Raspberry Pi single-board computer (SBC); and then start up the device.
SBCs are becoming ubiquitous-examples are Nest thermostats and Ring doorbells.
In later assignments you will configure and add software applications to the Raspberry Pi.

After completing this assignment you will be able to:

  1. Obtain latest Raspbian operating system
  2. Create a bootable micro-SD card containing the operating system
  3. Boot the Raspberry Pi
  4. Connect to the WSU Guest wireless network
  5. Configure the Raspberry Pi
  6. Use a web browser and other software on the Raspberry Pi
  7. Install an application on the Raspberry Pi to take screen shots
  8. Upload the screen shot to D2L.

Exercise 1-Obtain Operating System
  1. Open a browser to: https://www.balena.io/etcher/
  2. Download and save Etcher to your ' ' folder.
  3. Use File Explorer, find your Etcher install file, and double-click to install Etcher.
  4. Open a browser to: https://www.raspberrypi.org/downloads/raspbian/.
  5. Download the latest ZIP file of the Raspbian Operating System with desktop
    Raspbian With Desktop
  6. Save the ZIP file to your ' ' folder.
Exercise 2-Flash microSD card
  1. Install the microSD card into the microSD card slot on the right side of your laptop.
  2. Newer laptops without microSD slots will need to see the professor or a TA to get a microSD card reader.
  3. Open Etcher and make sure you see that it recognizes that you have a microSD card mounted.
    If Ether does not appear, you will need to restart your laptop.
  4. Select the ZIP file that your downloaded earlier and Drag and Drop it into etcher where it says select image.
    Flash Raspbian to microSD
  5. Then select Flash. The process will take several minutes.
Exercise 3-Boot Raspberry Pi
  1. While the microSD card is flashing, locate a USB keyboard and mouse.
    Plug them into the Pi at your station (We may not have keyboards and mice for everyone so please share).
  2. Once the flash is complete, remove the microSD card and put it into the Pi.
    Installing microSD into the Pi
  3. Press the power button on the Pi, which should be on top or around the monitor in front of you. You should see a red light on the Pi.
  4. On your monitor press the button with the colored dot.
    Note that pressing this button repeatedly swithches the monitor video signal between your laptop and the Raspberry Pi.
  5. Follow the on-screen instructions to set your country, language and Time-zone.
    Check the boxes for 'Use English language' and 'Use US keyboard'.
  6. Open Notepad, choose a password (Please use MagicMirror%).
    Save your password in a file named 'Readme.txt'; in your ' folder.
  7. If you forget your password you can change it under the settings.
  8. Check “This screen shows a black border around the desktop” if you have a black border.
  9. CONNECT TO WSU GUEST ONLY!!! If prompted make an account and log in.
  10. When asked to update, click next. The update will begin, this make take a few minutes. If asked to restart, do so.
  11. In the upper right corner of the desktop hover over the Wi-Fi signal to determine the IP address.
  12. Store this IP address in the ReadMe.txt file that contains your password.
    The IP address will look something like 10.20.xx.xx
    Note that the IP address will change about every 30 days.
  13. In the upper left corner, you will see the raspberry icon. click the raspberry icon, then preferences, and Raspberry Pi Configuration.
  14. Click the interfaces tab and Enable SSH and VNC.
Exercise 4-Screen Shots
  1. On the Raspberry Pi open a terminal window.
  2. In the Terminal window type sudo apt-get install scrot.
    Press Enter and the installation will begin.
  3. After scrot is installed, in the Terminal window type scrot -h
    This brings up information on how to use scrot.
    Review this information.
    Note that simply typing scrot in the Terminal window takes a scrot (short for SCReen shOT) and stores it in your Raspberry Pi's  '/home/pi' folder.
    scrot help
